# Web App

Next.js/React frontend for the ElevenLabs Wrapped experience.

- `src/app` – routing (public landing, protected wrap flow, API routes)
- `src/features` – vertical slices (auth, wrap story, generation, library, profile)
- `src/shared` – cross-cutting utilities, providers, hooks, and UI primitives
- `src/entities` – domain models and data mappers
- `src/styles` – design tokens and global styles
- `src/tests` – frontend test setup
