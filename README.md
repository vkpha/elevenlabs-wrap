# elevenlabs-wrap

Padel enjoyers ElevenLabs hackathon project.

## Structure

- `apps/web` – Next.js/React client for the Wrapped experience.
- `apps/api` – BFF for auth, stats, and ElevenLabs music generation.
- `packages` – Shared UI, SDKs, analytics, tooling configs, and types.
- `infrastructure` – IaC, CI/CD, and secrets templates.
- `scripts` – Dev/ops utilities.
- `docs` – Architecture notes and ADRs.

See per-folder READMEs for more detail.
